## 六、AI 分類實作

### 6.1 Prompt 設計

```python
CLASSIFICATION_PROMPT = """
請將以下文章分類到這些類別之一，並給出信心分數：

類別定義：
1. 教學型 - 有步驟、方法的教學內容
2. 經驗分享 - 個人經歷、案例分享
3. 觀點評論 - 思考、評論、觀點
4. 資源整理 - 工具、清單、推薦
5. 問答互動 - 提問、徵求意見
6. 成果展示 - 里程碑、成就展示
7. 生活記錄 - 日常、心情、隨筆
8. 其他 - 無法明確分類

文章內容：
{text}

請以 JSON 格式回傳：
{{
  "category": "類別名稱",
  "confidence": 0.95,
  "reasoning": "簡短說明分類理由"
}}
"""
```

### 6.2 實作程式碼

```python
import openai

def classify_article(text):
    """使用 GPT-4 分類文章"""
    response = openai.ChatCompletion.create(
        model="gpt-4",
        messages=[
            {"role": "system", "content": "你是一個專業的內容分類專家。"},
            {"role": "user", "content": CLASSIFICATION_PROMPT.format(text=text)}
```
