## 三、詞彙分類分析

### 3.1 主題分類定義

**分類標準：**

1. **創業/商業**
   - 關鍵詞：創業、公司、營收、商業模式、獲利...
   - 特徵：商業策略、創業經驗

2. **自媒體/行銷**
   - 關鍵詞：流量、粉絲、變現、演算法、互動...
   - 特徵：社群經營、內容策略

3. **工具/技術**
   - 關鍵詞：AI、Notion、自動化、效率、工具...
   - 特徵：工具教學、技術應用

4. **個人成長**
   - 關鍵詞：學習、思考、習慣、目標、改變...
   - 特徵：成長心態、自我提升

5. **工作/職涯**
   - 關鍵詞：工作、公司、主管、同事、離職...
   - 特徵：職場經驗、職涯規劃

6. **生活/心態**
   - 關鍵詞：生活、平衡、焦慮、壓力、休息...
   - 特徵：生活感悟、心態調整

7. **副業/斜槓**
   - 關鍵詞：副業、兼職、被動收入、斜槓...
   - 特徵：多元收入、時間管理

8. **其他**
   - 無法歸類的詞彙

## 四、詞彙特徵分析

### 4.1 高互動詞彙特徵

**發現的模式：**

1. **趨勢詞**
   - 特徵：當下熱門話題
   - 例如：AI、ChatGPT、遠端工作
   - 互動：高按讚 + 高回覆

2. **痛點詞**
   - 特徵：引發共鳴的問題
   - 例如：焦慮、疲憊、迷茫
   - 互動：高回覆 + 高轉發

3. **方法論詞**
   - 特徵：實用性強的技巧
   - 例如：步驟、方法、技巧、模板
   - 互動：高按讚 + 高轉發

4. **數字詞**
   - 特徵：具體可量化
   - 例如：5個、3步驟、10倍
   - 互動：高點擊 + 高保存

## 五、內容策略建議

### 5.1 關鍵詞使用建議

**高互動關鍵詞清單：**
```
必用詞（Top 10）：
1. [待分析]
2. [待分析]
...

選用詞（Top 20）：
1. [待分析]
2. [待分析]
...
```

### 5.2 內容組合策略

**推薦組合 A：趨勢教學**
```
關鍵詞：AI + 工具 + 步驟
範例：「5個AI工具，讓你工作效率提升10倍」
預期互動：高按讚 + 高轉發
```

**推薦組合 B：痛點共鳴**
```
關鍵詞：焦慮 + 方法 + 改變
範例：「工作焦慮怎麼辦？3個方法幫你改變現狀」
預期互動：高回覆 + 高共鳴
```

**推薦組合 C：經驗分享**
```
關鍵詞：創業 + 經驗 + 數字
範例：「創業1年，從0到月收10萬的3個關鍵」
預期互動：高按讚 + 高保存
```

### 5.3 避雷指南

**低互動特徵：**
- ❌ 過於專業的術語（讀者看不懂）
- ❌ 負面情緒詞彙（抱怨、批評）
- ❌ 模糊不清的詞（可能、或許、大概）

## 六、實作指南

### 6.1 中文斷詞工具

**推薦工具：**

1. **jieba** (簡單易用)
```python
import jieba

text = "你的文章內容"
words = jieba.cut(text)
```

2. **ckiptagger** (學術級精準)
```python
from ckiptagger import WS

ws = WS("./data")
words = ws([text])
```

### 6.2 自定義詞典

建議新增專有名詞：
```
創業圈.txt:
- 一人公司
- 自媒體
- 數位遊牧
- 被動收入
- 知識變現
```

### 6.3 分析流程

```python
# 1. 載入資料
df = pd.read_csv('scripts/analyzed_data.csv')

# 2. 斷詞
df['words'] = df['text'].apply(segment_words)

# 3. 詞頻統計
word_freq = Counter()
for words in df['words']:
    word_freq.update(words)

# 4. 互動分析
word_engagement = df.groupby('words')['like_count'].mean()

# 5. 分類統計
categorize_words(word_freq)
```
